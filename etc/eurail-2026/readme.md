## 技術について

MapLibre GL JSをGISエンジンとして使っています。
ベースマップはMapTiler経由でOpenStreetMapのJPタイルを呼び出していて、
国境データはMapLibre Demo TilesをVectorとして読み込んで選択可能にしています。👉 オーバーレイさせる方法がよくわからなかったので、国境と独立して`Natural Earth Data`から抽出したgeojsonを重ねています(一部重なっていない部分が見えると思います。)

路線図としてOpenRailwayMapがタイルとして呼び出せることをClaudeくんが教えてくれたので試してみました。
